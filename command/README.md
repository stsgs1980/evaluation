# Команды

Справка по всем доступным командам opencode: встроенным и пользовательским.

## Встроенные команды

Доступны в любом проекте без настройки.

| Команда | Горячая клавиша | Назначение |
|---------|----------------|-----------|
| `/init` | -- | Настройка `AGENTS.md` (пошаговый мастер) |
| `/connect` | -- | Добавить AI-провайдер (API-ключ) |
| `/models` | `ctrl+x m` | Список доступных моделей |
| `/new` | `ctrl+x n` | Начать новую сессию (alias: `/clear`) |
| `/sessions` | `ctrl+x l` | Список сессий, переключение (alias: `/resume`, `/continue`) |
| `/undo` | `ctrl+x u` | Отменить последнее сообщение + откат файлов через git |
| `/redo` | `ctrl+x r` | Повторить отменённое сообщение + восстановить файлы |
| `/share` | -- | Поделиться ссылкой на текущую сессию |
| `/unshare` | -- | Убрать сессию из публичного доступа |
| `/compact` | `ctrl+x c` | Сжать текущую сессию (alias: `/summarize`) |
| `/details` | -- | Показать/скрыть детали выполнения инструментов |
| `/editor` | `ctrl+x e` | Открыть внешний редактор для набора сообщения |
| `/export` | `ctrl+x x` | Экспорт сессии в Markdown и открыть в редакторе |
| `/themes` | `ctrl+x t` | Список доступных тем оформления |
| `/thinking` | -- | Показать/скрыть блоки рассуждений модели (extended thinking) |
| `/help` | -- | Справка по командам |
| `/exit` | `ctrl+x q` | Выйти из opencode (alias: `/quit`, `/q`) |

Также доступны: `!команда` для запуска shell-команд и `@файл` для включения файла в контекст.

## Пользовательские команды

Файлы в `command/`. Вызываются по имени: `/commit`, `/review-pr` и т.д.

| Команда | Назначение |
|---------|-----------|
| `commit` | Создать git-коммит из staged-изменений |
| `commit-push-pr` | Коммит + push + открыть PR (полный цикл) |
| `code-review` | Ревью своего PR (код, тесты, документация) |
| `review-pr` | Ревью чужого PR специализированными агентами |
| `configure` | Включение/отключение правил hookify |
| `create-plugin` | Пошаговое создание нового плагина |
| `feature-dev` | Разработка фичи с анализом кодовой базы |
| `hookify` | Создание хуков для предотвращения нежелательных действий |
| `list` | Показать все настроенные правила hookify |
| `clean_gone` | Удалить все git-ветки со статусом `[gone]` |
| `revise-claude-md` | Обновить CLAUDE.md по итогам сессии |
| `help` | Справка по командам и плагинам |
